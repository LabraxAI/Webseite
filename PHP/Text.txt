<b><u>Was ist PHP:</u></b>
PHP( Hypertext Preprocessor, urspr&uuml;nglich Personal Home Page Tools)ist eine Skriptsprache die haupts&auml;chlich zur Erstellung dynamischer Webseiten verwendet wird dies bedeutet
das eine Komonikation mit der Webseite m&ouml;glich z.B. in Form eines Chat auch gibt es Funktinen die dem Besitzer der Website erlauben z.B. die Anzahl von Besuchern zu  Z&aum;len.r>
<br>
<br>
<b><u>Wie Funktionirt es:</u></b><br>
Um PHP anwenden zu können mus der PHP-code in einer PHP Datei geschrieben werden  diese PHP-Datei funktionrt wie eine Normale HTML-Datei nur das auch PHP-code verwendet werden kann
der PHP-code wird dann in einen solchen tag geschrieben <?php...(code)...?>.<br>
Damit dies funktionirt muss die Datei jedoch auf einem Server liegen der PHP unterst&uuml;tzt zudem sind weitere kleinigkeiten zu beachten wie z.B. das eine Variable immer mit einem   $-Zeichen Beginnt und das ,dass Gleichzeichen nicht im Mathematischen sinne zu verstehen ist sondern als Zuweisung gilt <br>
